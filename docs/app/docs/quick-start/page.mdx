
# 快速开始

本指南将帮助你快速上手 Skyroc UI。

## 创建项目

首先，创建一个新的 React 项目（如果你还没有的话）：

```bash
# 使用 Vite
npm create vite@latest my-app -- --template react-ts
cd my-app

# 或使用 Next.js
npx create-next-app@latest my-app
cd my-app
```

## 安装依赖

```bash
npm install @skyroc/ui
npm install -D tailwindcss postcss autoprefixer
```

## 配置 Tailwind

初始化 Tailwind CSS：

```bash
npx tailwindcss init -p
```

更新 `tailwind.config.js`：

```js
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    './index.html',
    './src/**/*.{js,ts,jsx,tsx}',
    './node_modules/@skyroc/ui/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {
      colors: {
        brand: 'hsl(237, 100%, 70%)',
      },
    },
  },
  plugins: [],
}
```

在你的主 CSS 文件中添加：

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

## 使用组件

现在你可以开始使用组件了：

```tsx
import { Button, Card } from '@skyroc/ui';

export default function App() {
  return (
    <div className="min-h-screen flex items-center justify-center p-8">
      <Card className="max-w-md p-6">
        <h1 className="text-2xl font-bold mb-4">欢迎使用 Skyroc UI</h1>
        <p className="text-muted-foreground mb-6">
          这是一个使用 Skyroc UI 构建的示例应用。
        </p>
        <div className="flex gap-3">
          <Button variant="primary">主要按钮</Button>
          <Button variant="outline">次要按钮</Button>
        </div>
      </Card>
    </div>
  );
}
```

## 主题配置

Skyroc UI 支持深色模式。你可以使用 `next-themes` 或其他主题切换方案：

```bash
npm install next-themes
```

```tsx
import { ThemeProvider } from 'next-themes'

function App({ Component, pageProps }) {
  return (
    <ThemeProvider attribute="class" defaultTheme="system">
      <Component {...pageProps} />
    </ThemeProvider>
  )
}
```
