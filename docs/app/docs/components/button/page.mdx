# Button 按钮

按钮用于开始一个即时操作。

## 导入

```tsx
import { Button } from '@skyroc/ui';
```

## API

### Button Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| variant | `'default' | 'primary' | 'destructive' | 'outline' | 'secondary' | 'ghost' | 'link'` | `'default'` | 按钮样式变体 |
| size | `'default' | 'sm' | 'lg' | 'icon'` | `'default'` | 按钮尺寸 |
| disabled | `boolean` | `false` | 是否禁用 |
| className | `string` | - | 自定义类名 |
| children | `React.ReactNode` | - | 按钮内容 |

## 最佳实践

### 语义化

使用合适的变体来表达不同的操作语义：

- **Primary**：主要操作，每个区域最多一个
- **Default**：次要操作
- **Destructive**：危险操作，如删除、注销
- **Ghost**：轻量级操作
- **Link**：导航类操作

### 无障碍

- 确保按钮有明确的文本说明
- 为图标按钮添加 `aria-label`
- 使用 `disabled` 而非视觉样式来禁用按钮

### 性能

按钮组件已经过优化，但在列表中大量使用时，考虑使用 `React.memo` 包裹。
