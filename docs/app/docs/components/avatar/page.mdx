# Avatar 头像

用于展示用户头像图片，支持图片、文字和备用方案。

## 导入

```tsx
import { Avatar, AvatarImage, AvatarFallback } from '@skyroc/ui';
```

## 基础用法

<ComponentPreview code={`import { Avatar, AvatarImage, AvatarFallback } from '@skyroc/ui';

export default function Demo() {
  return (
    <Avatar>
      <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
      <AvatarFallback>CN</AvatarFallback>
    </Avatar>
  );
}`}>
  <div className="relative flex h-10 w-10 shrink-0 overflow-hidden rounded-full">
    <div className="flex h-full w-full items-center justify-center rounded-full bg-muted">
      <span className="text-sm font-medium">CN</span>
    </div>
  </div>
</ComponentPreview>

## API

### Avatar

头像容器组件。

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| className | `string` | - | 自定义类名 |

### AvatarImage

头像图片组件，继承所有 img 元素属性。

### AvatarFallback

头像备用内容，当图片加载失败时显示。
