# Card 卡片

通用卡片容器，可承载文字、列表、图片、段落等内容。

## 导入

```tsx
import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter } from '@skyroc/ui';
```

## API

### Card Props

基础卡片容器组件。

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| className | `string` | - | 自定义类名 |
| children | `React.ReactNode` | - | 卡片内容 |

### CardHeader

卡片头部区域。

### CardTitle

卡片标题。

### CardDescription

卡片描述文字。

### CardContent

卡片主要内容区域。

### CardFooter

卡片底部区域，通常放置操作按钮。
