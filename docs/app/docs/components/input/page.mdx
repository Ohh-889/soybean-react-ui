# Input 输入框

通过键盘输入内容的基本表单组件。

## 导入

```tsx
import { Input } from '@skyroc/ui';
```

## API

### Input Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| type | `string` | `'text'` | 输入框类型 |
| placeholder | `string` | - | 占位文本 |
| disabled | `boolean` | `false` | 是否禁用 |
| className | `string` | - | 自定义类名 |

继承所有原生 HTML input 元素的属性。
