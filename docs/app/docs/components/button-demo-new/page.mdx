## 🎯 实现原理

这个页面使用了 **编译时转换** 方案：

### 1. 写 MDX
```mdx
<Demo src="@/demos/button-basic.tsx" title="基础按钮" />
```

### 2. rehype 插件自动转换为
```tsx
import ButtonBasic from '@/demos/button-basic.tsx';

<Demo src="@/demos/button-basic.tsx" title="基础按钮">
  <ButtonBasic />
</Demo>
```

### 3. 优势
- ✅ **零运行时开销** - 无需动态 import
- ✅ **静态可分析** - Webpack/Turbopack 友好
- ✅ **完整类型支持** - 编译时检查
- ✅ **自动代码分割** - 每个 demo 独立 chunk
- ✅ **简单易用** - 只需提供文件路径

这就是 **dumi** 的核心理念！🚀
