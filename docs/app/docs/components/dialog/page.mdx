# Dialog 对话框

在保留当前页面状态的情况下，打开一个浮层对话框，承载相关操作。

## 导入

```tsx
import {
  Dialog,
  DialogTrigger,
  DialogContent,
  DialogHeader,
  DialogTitle,
  DialogDescription,
  DialogFooter
} from '@skyroc/ui';
```

## 基础用法

<ComponentPreview code={`import {
  Dialog,
  DialogTrigger,
  DialogContent,
  DialogHeader,
  DialogTitle,
  DialogDescription
} from '@skyroc/ui';
import { Button } from '@skyroc/ui';

    <Dialog>

  </button>

## API

### Dialog

对话框根组件。

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| open | `boolean` | - | 控制对话框打开状态 |
| onOpenChange | `(open: boolean) => void` | - | 打开状态改变时的回调 |

### DialogTrigger

触发对话框打开的按钮。

### DialogContent

对话框内容容器。

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| className | `string` | - | 自定义类名 |

### DialogHeader

对话框头部区域。

### DialogTitle

对话框标题。

### DialogDescription

对话框描述文字。

### DialogFooter

对话框底部区域，通常放置操作按钮。
